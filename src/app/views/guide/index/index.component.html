5. WebStorm 설정

GIT 설정

WebStorm을 실행한다.

하단의 Configuration내의 Settings를 선택한다.

Settings 내의 Version Control로 이동하여 GIT를 선택한 뒤, Path to Git executable에 설치한 GIT.exe 경로를 지정한다.

소스 받기

WebStorm을 실행한 뒤, Check out from Version Control를 선택한다.

BitBucket URL과 저장될 디렉터리를 지정한 후 CLONE 을 한다.

6. 실행

라이브러리 설치

CLONE 받은 디렉터리로 이동한다. 단, 위 그림과 같이 package.json이 있는 경로까지로 이동한다.

npm install 명령어로 라이브러리를 설치한다.

실행

ng serve -o 명령어로 프로젝트를 실행한다. -o는 프로젝트가 시작되면, 기본 브라우저로 자동으로 보여주는 옵션이다.

빌드

ng build --prod --env=prod 명령어로 빌드한다. --prod는 production 모드로 빌드를 한다는 옵션이며, --env=는 어떤 환경의 값으로 빌드할 건지 지정하는 옵션이다.
